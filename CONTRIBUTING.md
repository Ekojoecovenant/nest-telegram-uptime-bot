# Contributing to Telegram Uptime Monitor Bot

Thank you for considering contributing!

## How to Contribute

1. **Fork** the repository
2. Create a new branch for you feature/bugfix:

    ```bash
    git checkout -b feature/my-cool-feature
    ```

3. Make your changes
4. Commit with clear messages:

    ```text
    feat: add delete confirmation
    fix: handle empty list state
    ```

5. Push to your fork
6. Open a Pull Request to the main repo

    - Describe what you changed and why
    - Reference any related issues

## Guidelines

- Follow existing code style (NestJS conventions TypeScript strict)
- Add comments for non-obvious logic
- Test locally before submitting
- Keep PRs focused (one feature/fix per PR)
- Update README if you change user-facing behavior
